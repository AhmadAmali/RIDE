```python
from pydantic import BaseModel, Field

class ObligationItem(BaseModel):
    """A single compliance obligation extracted from a regulatory document."""
    text: str = Field(description="Clear summary of the obligation in your own words")
    source_quote: str = Field(description="Verbatim text from the document that establishes this obligation")
    reasoning: str = Field(description="2-3 sentences explaining WHY this text creates a compliance requirement")
    is_ambiguous: bool = Field(description="True if language could be interpreted multiple ways or has unclear scope")

class ObligationList(BaseModel):
    """List of obligations extracted from a document chunk."""
    obligations: list[ObligationItem]
```

Define the extraction prompt as a module-level constant `EXTRACTION_PROMPT`:
```python
EXTRACTION_PROMPT = """You are a compliance analyst reviewing a regulatory document.

Analyze the following excerpt and identify all compliance obligations -- requirements, prohibitions, or duties that an organization must fulfill.

For each obligation you identify:
1. Write a clear summary in your own words (text field)
2. Copy the EXACT verbatim text from the document that establishes this obligation (source_quote field). The quote must appear word-for-word in the excerpt below.
3. Explain in 2-3 sentences WHY this text creates a compliance requirement -- what regulatory principle it enforces and what the consequence of non-compliance would be (reasoning field). Be substantive: do not just say "this is an obligation."
4. Set is_ambiguous to true if the language could be interpreted multiple ways, has unclear scope, or requires human judgment to apply correctly (is_ambiguous field)

Only include genuine obligations. Informational statements, definitions, and recitals are NOT obligations.

If no obligations are found in this excerpt, return an empty obligations list.

Document excerpt:
---
{chunk}
---"""
```

Define the chunking helper function:
```python
def chunk_markdown(markdown: str, chunk_chars: int = 16000, overlap_chars: int = 1600) -> list[str]:
    """Split Markdown into overlapping character windows.

    16,000 chars ~ 4,000 tokens. 10% overlap prevents splitting obligations at boundaries.
    Returns at least one chunk even for short documents.
    """
    if len(markdown) <= chunk_chars:
        return [markdown]
    chunks = []
    start = 0
    while start < len(markdown):
        end = start + chunk_chars
        chunks.append(markdown[start:end])
        start += chunk_chars - overlap_chars
    return chunks
```

All three (models, prompt, chunking) live in the same file because they form a single extraction contract used only by the extract worker.

- Implement `async def process(self, message: dict) -> None:`:
  1. Extract `doc_id = message["document_id"]`
  2. Fetch document from DB: `async with async_session_maker() as session:` -> `select(Document).where(Document.id == uuid.UUID(doc_id))` -> get `doc.content_markdown`
  3. If `content_markdown` is None or empty, log a warning and return (no-op -- the document was not parsed successfully)
  4. Chunk the markdown: `chunks = chunk_markdown(doc.content_markdown)`
  5. For each chunk, call Claude in a thread (the Anthropic SDK is synchronous): `response = await asyncio.to_thread(self._extract_chunk, chunk)` where `_extract_chunk` is a sync helper method
  6. `_extract_chunk(self, chunk_text: str) -> list[ObligationItem]`:
     - Call `response = self._client.messages.parse(model="claude-sonnet-4-5", max_tokens=4096, messages=[{"role": "user", "content": EXTRACTION_PROMPT.format(chunk=chunk_text)}], output_format=ObligationList)`
     - Return `response.parsed_output.obligations`
  7. Collect all obligations from all chunks into a flat list. Use deduplication: if two ObligationItems from overlapping chunks have a `source_quote` that shares >80% character overlap (using simple substring containment check), keep only the first. A simple approach: for each new obligation, check if any already-collected obligation's source_quote contains it as a substring or vice versa. If so, skip the duplicate.
  8. Persist obligations to DB: `async with async_session_maker() as session:` -> for each ObligationItem, create an `Obligation(document_id=uuid.UUID(doc_id), text=item.text, source_quote=item.source_quote, reasoning=item.reasoning, is_ambiguous=item.is_ambiguous, status="pending")`, `session.add(obligation)`. Then update `doc.status = "extracted"` (re-fetch doc in same session). `await session.commit()`.
  9. Emit `KafkaTopic.OBLIGATION_EXTRACTED` with payload `{"document_id": doc_id, "obligation_count": len(all_obligations)}`
