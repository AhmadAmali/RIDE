**1. GET /api/obligations?document_id={uuid}**
- Query param: `document_id` (required UUID)
- Returns list of obligations for the given document, ordered by `created_at`
- Response: list of dicts with id, document_id, text, source_quote, reasoning, status, is_ambiguous, created_at
- Uses `get_db` dependency injection (same pattern as documents.py)

**2. PATCH /api/obligations/{obligation_id}/review**
- Path param: `obligation_id` (UUID)
- Request body JSON: `{"action": "approved" | "rejected", "actor": "legal-reviewer"}`
- Create a Pydantic model `ReviewRequest` with `action: Literal["approved", "rejected"]` and `actor: str = "legal-reviewer"`
- **CRITICAL: Atomic transaction** — use `async with session.begin():` to wrap BOTH the obligation status update AND the AuditLog insert in a single transaction. Do NOT use two separate commits. Pattern from research:
  ```python
  async with session.begin():
      result = await session.execute(select(Obligation).where(Obligation.id == obligation_id))
      obligation = result.scalar_one_or_none()
      if not obligation:
          raise HTTPException(404)
      if obligation.status != "pending":
          raise HTTPException(409, detail="Obligation already reviewed")
      obligation.status = body.action
      session.add(AuditLog(
          entity_type="obligation",
          entity_id=obligation_id,
          action=body.action,
          actor=body.actor,
          metadata_={"previous_status": "pending"},
      ))
  ```
- After the transaction commits successfully, emit Kafka event. Only emit for "approved" actions:
  ```python
  if body.action == "approved":
      await request.app.state.kafka_producer.send(
          KafkaTopic.OBLIGATION_APPROVED,
          {"obligation_id": str(obligation_id), "document_id": str(obligation.document_id)},
      )
  ```
- Return the updated obligation as a dict

**3. GET /api/obligations/{obligation_id}**
- Return single obligation by ID (for frontend detail view)
- 404 if not found

     ```python
                 system_name=result.metadata.get("service", "unknown"),

                 suggested_by="rag",

             )
             session.add(mapping)
         await session.commit()
     ```

```python
@asynccontextmanager
async def lifespan(app: FastAPI):
    # --- Startup ---
    # 1. Shared Kafka producer (existing)
    producer = BaseProducer()
    await producer.start()
    app.state.kafka_producer = producer

    # 2. Qdrant client — shared between corpus indexer and RAG mapper
    qdrant_client = AsyncQdrantClient(url=settings.qdrant_url)
    app.state.qdrant_client = qdrant_client

    # 3. Index corpus at startup (idempotent — skips if already indexed)
    await index_corpus(qdrant_client, str(Path(__file__).resolve().parents[3] / "data" / "corpus"))

    # 4. Parse worker (existing)
    parse_worker = ParseWorker()
    await parse_worker.start()
    parse_task = asyncio.create_task(parse_worker.run())
    app.state.parse_worker = parse_worker
    app.state.parse_task = parse_task

    # 5. Extract worker (existing)
    extract_worker = ExtractWorker()
    await extract_worker.start()
    extract_task = asyncio.create_task(extract_worker.run())
    app.state.extract_worker = extract_worker
    app.state.extract_task = extract_task

    # 6. Action item worker (NEW)
    action_item_worker = ActionItemWorker()
    await action_item_worker.start()
    action_item_task = asyncio.create_task(action_item_worker.run())
    app.state.action_item_worker = action_item_worker
    app.state.action_item_task = action_item_task

    # 7. RAG mapper worker (NEW) — receives qdrant_client
    rag_mapper_worker = RagMapperWorker(qdrant_client=qdrant_client)
    await rag_mapper_worker.start()
    rag_mapper_task = asyncio.create_task(rag_mapper_worker.run())
    app.state.rag_mapper_worker = rag_mapper_worker
    app.state.rag_mapper_task = rag_mapper_task

    yield

    # --- Shutdown (reverse order) ---
    rag_mapper_task.cancel()
    await rag_mapper_worker.stop()
    action_item_task.cancel()
    await action_item_worker.stop()
    extract_task.cancel()
    await extract_worker.stop()
    parse_task.cancel()
    await parse_worker.stop()
    await qdrant_client.close()
    await producer.stop()
    await engine.dispose()
```
